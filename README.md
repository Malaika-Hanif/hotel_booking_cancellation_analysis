
#  Hotel Booking Cancellation Analysis

##  Project Overview

This project analyzes hotel booking data to understand the factors that influence booking cancellations.
The goal is to extract meaningful insights using **Data Cleaning, Exploratory Data Analysis (EDA), and Visualization** techniques.

This project demonstrates core Data Analyst skills including:

* Data Cleaning
* Handling Missing Values
* Outlier Detection
* Feature Engineering
* GroupBy Analysis
* Data Visualization
* Insight Generation

---

##  Dataset Information

* Dataset: Hotel Booking Dataset
* Rows: 119,000+
* Features: 30+ columns
* Target Variable: `is_canceled`

### Key Features:

* `hotel`
* `lead_time`
* `arrival_date_year`
* `arrival_date_month`
* `stays_in_weekend_nights`
* `stays_in_week_nights`
* `adults`, `children`
* `country`
* `adr` (Average Daily Rate)
* `is_canceled`

##  Key Insights

* Higher lead time increases cancellation probability.
* City hotels have higher cancellation rates than resort hotels.
* Certain countries show higher booking volume.
* Price (ADR) impacts cancellation behavior.
* Most cancellations occur within specific months.

---

##  Business Recommendations

* Offer discounts for long lead-time bookings.
* Introduce stricter cancellation policies for high-risk months.
* Focus marketing campaigns on high-performing countries.
* Optimize pricing strategy based on demand trends.
